# Fluxo - Frontend (React + Vite)

Interface simplificada para gestão de estoque de produtos, permitindo cadastro, busca, edição e retirada de itens.

## Tecnologias
- **React** (com TypeScript)
- **Axios** (Integração com API)
- **Vite** (Build tool)

## Como rodar
1. Certifique-se de que o arquivo `.env` está na raiz da pasta `web` com a URL da API:
   ```env
   VITE_API_URL=http://localhost:5243/api
